## Project Title
Personal Portfolio Website

## Objective
To create a responsive personal portfolio website that showcases my skills, projects, and professional experience using modern web technologies.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome (for icons)
- Google Fonts

## Project Description
This portfolio website serves as a digital resume and project showcase. It is designed with a mobile-first approach to ensure optimal viewing experience across all devices. The website includes the following sections:

1. **Home/Hero Section**: Introduction with a call-to-action
2. **About Section**: Personal information, education, and experience
3. **Skills Section**: Technical skills displayed in an interactive grid
4. **Projects Section**: Showcase of recent projects with descriptions and links
5. **Contact Section**: Contact form and personal contact information
6. **Footer**: Copyright information and social media links

## Implementation Details

### Responsive Design
The website uses CSS media queries to adapt to different screen sizes:
- Mobile (< 576px)
- Tablet (576px - 768px)
- Desktop (> 768px)

### Performance Optimization
- Optimized images for faster loading
- Minimal use of external libraries
- Efficient CSS selectors

### Accessibility Features
- Semantic HTML elements
- ARIA attributes where necessary
- Keyboard navigation support
- Sufficient color contrast
